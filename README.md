# Windows RT Installer for Lumia Devices

A script to install Windows RT on Lumia devices.

## Prerequisites

- **Install.wim file**: You must obtain an `Install.wim` file from a Windows RT ISO image
- Place the `Install.wim` file in the same directory as the installer script

## Usage

1. Download a Windows RT ISO image
2. Extract the `Install.wim` file from the ISO
3. Place `Install.wim` in the same folder as this installer script
4. Run the installer script
5. Follow the on-screen prompts to complete the installation

## Requirements

- Compatible Lumia device
- USB cable for device connection
- Windows RT ISO image (source for Install.wim)

## Notes

- Ensure your device is fully charged before starting the installation
- Do not disconnect the device during the installation process
- Back up your device data before proceeding
